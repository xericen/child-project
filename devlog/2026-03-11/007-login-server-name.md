# 로그인 페이지 어린이집 이름 동적 표시

- **ID**: 007
- **날짜**: 2026-03-11
- **유형**: 기능 추가

## 작업 요약
로그인 페이지(page.main)의 타이틀을 "child Welcome!"에서 세션에 저장된 어린이집 이름으로 동적 변경.
예: A어린이집 서버 참여 시 "A어린이집 Welcome!", B어린이집 서버 참여 시 "B어린이집 Welcome!"

## 변경 파일 목록
### 프론트엔드
- `src/app/page.main/view.pug`: 타이틀을 `{{ serverName || 'child' }}` 바인딩으로 변경, 기존 서버 배지(server-badge) 제거 (타이틀에 이미 서버 이름 표시)
