# 알레르기 DB 카테고리 및 아동 데이터 정리

- **ID**: 007
- **날짜**: 2026-03-17
- **유형**: 버그 수정

## 작업 요약
토리(child_id=15)의 알레르기 데이터가 잘못 등록(땅콩, 기타(생선))되어 있어 올바른 데이터(기타(굴), 기타(키위))로 수정. AllergyCategories 테이블의 카테고리별 주의식품은 이미 HWP 기준으로 정확히 반영되어 있음을 확인.

## 변경 내역

### DB 데이터 수정
- `child_allergies`: child_id=15 (토리) 기존 데이터(id=31: 땅콩, id=32: 기타(생선)) 삭제
- `child_allergies`: child_id=15 (토리) 새 데이터 추가 — 기타(굴) [어류 및 갑각류], 기타(키위) [과채류]

### 확인 사항
- `allergy_categories`: 난류(달걀, 알부민, 마요네즈 등), 우유류, 견과류, 대두, 곡류(메밀), 어류 및 갑각류(굴 포함), 고기류, 과채류(키위 포함) — 모두 HWP 기준과 일치
