# 알레르기 경고 정확도 개선 — 오늘 식단에 없는 알레르기 제외

- **ID**: 001
- **날짜**: 2026-03-17
- **유형**: 버그 수정

## 작업 요약
`기타` 알레르기 타입(굴, 키위, 생선 등)이 표준 19종 번호에 매핑되지 않아 오늘 식단에 해당 음식이 없어도 무조건 경고가 표시되던 문제 수정. 매핑 불가한 키워드는 오늘 식단 메뉴명에서 직접 검색하여 실제 포함된 경우에만 경고 표시하도록 변경.

## 변경 파일 목록
### API
- `src/app/page.note.meal/api.py`: `get_daily()` 함수의 알레르기 경고 로직 재작성
  - `all_menu_text` 변수 추가 (당일 전체 메뉴명 결합)
  - `기타` 타입: 1차로 `ALLERGY_TYPE_TO_NUMBERS` 매핑 시도, 실패 시 메뉴명에서 키워드 직접 검색
  - 매칭된 음식이 있을 때만 `allergy_alerts`에 추가
