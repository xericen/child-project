# 영양 평가 기준 및 실무 경고 구조화

- **ID**: 004
- **날짜**: 2026-04-14
- **유형**: 기능 추가

## 작업 요약
어린이집 식단 평가 목적에 맞춰 탄수화물/단백질/지방 비율, 단백질 총량, 나트륨 목표치를 계산하는 평가 로직을 추가했다.
국+김치 동시 제공, 가공식품 포함, 빵/주스 간식, 간식 당류 과다 같은 실무 경고를 구조화하여 월간 영양 대시보드 응답에 포함했다.

## 변경 파일 목록

### Model
- `src/model/nutrition_api.py`
  - `evaluate_daycare_nutrition()` 추가
  - 탄단지 비율 55~65 / 10~20 / 20~30 계산 및 단백질 7g 이상, 나트륨 400~600mg 체크 구현
  - 국+김치, 가공식품, 고당류 간식 경고 규칙 추가

### API
- `src/app/page.note.meal.nutrition/api.py`
  - 일별 평가(`daily_evaluations`) 및 월간 종합 평가(`evaluation`)를 대시보드 응답에 추가
  - 분해 메뉴 개수(`decomposed_count`) 집계 추가
