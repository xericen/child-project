# 어린이 연령별 1인분 칼로리 계산 정확도 개선

- **ID**: 002
- **날짜**: 2026-04-06
- **유형**: 기능 개선

## 작업 요약
AI 저녁메뉴 추천이 아이의 정확한 나이(만 나이 + 개월수)를 반영하지 않고 일반적인 음식 기준으로만 동작하는 문제 개선. 연령 그룹(1~2세/3~5세) 분류는 유지하되, AI 프롬프트에 개월수·발달단계별 음식 가이드·1인분 기준 중량을 명시적으로 전달.

## 변경 파일 목록

### 백엔드 (api.py)
- `src/app/page.note.today/api.py`
  - `_get_my_children_info()`: `age_months`(총 개월수), `birthdate`(생년월일 문자열) 필드 추가
  - Stage 3 AI 프롬프트 전면 개선:
    - 정확한 만 나이 표시 (예: "만 1세 3개월")  
    - 발달단계별 음식 가이드 자동 생성 (≤18개월: 이유식 완료기, ≤30개월: 유아식, ≤3세: 유아식 완성기, 3~5세: 일반 유아)
    - 각 단계별 적절한 식감·조리법·1회 식사량 비율 명시
    - SERVING_WEIGHTS에서 연령별 1인분 기준 중량을 AI에 직접 전달
    - system_instruction에 "아이 나이의 1인분 기준으로 계산" 규칙 강제
