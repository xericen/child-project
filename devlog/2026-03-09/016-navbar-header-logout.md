# 네비바 헤더 UI + 로그아웃 기능 구현

- **ID**: 016
- **날짜**: 2026-03-09
- **유형**: 기능 추가

## 작업 요약
layout.navbar에 헤더 오버레이 구현. 로그인/회원가입/비밀번호찾기 페이지를 제외한 모든 페이지 우상단에 🔔 알림 벨 + ⋮ 더보기 아이콘 배치. 더보기 드롭다운에 회원정보/로그아웃 버튼 추가. 로그아웃 시 세션 삭제 후 로그인 페이지로 이동.

## 변경 파일 목록
### layout.navbar
- **app.json**: `"controller": "base"` 추가 (세션 접근용)
- **view.ts**: Router 구독으로 경로별 헤더 표시/숨김, 드롭다운/알림 토글, 로그아웃 처리
- **view.pug**: 헤더 오버레이(벨+더보기), 드롭다운 메뉴, 알림 패널(빈 상태), 배경 오버레이
- **view.scss**: 고정 위치 헤더, 아이콘 버튼, 배지, 드롭다운 메뉴 스타일
- **api.py**: `get_unread_count()` (FN-0020 대비 스텁), `logout()` (세션 클리어)
