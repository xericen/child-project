# note 페이지에 알레르기/식단표 버튼 UI 구현

- **ID**: 002
- **날짜**: 2026-03-09
- **유형**: 기능 추가

## 작업 요약
로그인 후 이동하는 /note 페이지에 기존 로그인/회원가입 페이지와 동일한 디자인 템플릿(둥근 카드, 보라색 테마)을 적용하고, 알레르기/식단표 2개 버튼을 세로로 배치하는 UI를 구현했다.

## 변경 파일 목록
### Page - page.note
- `src/app/page.note/view.pug`: 카드 레이아웃 + 알레르기/식단표 버튼 2개 세로 배치
- `src/app/page.note/view.ts`: Service 주입 및 초기화, 버튼 클릭 핸들러 추가
- `src/app/page.note/view.scss`: 기존 스타일 톤 적용 + `:host` 블록 선언
