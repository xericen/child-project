# AI 저녁 메뉴 추천 개인화 (알레르기 맞춤 + 나이별 영양)

- **ID**: 003
- **날짜**: 2026-03-20
- **유형**: 기능 추가

## 작업 요약
기존 `recommend_dinner()` 함수를 알레르기 맞춤 개인화 추천으로 개선했다. 부모가 저녁 추천을 요청하면 자녀의 알레르기 정보와 나이를 AI 프롬프트에 포함하여, 알레르기 식품을 제외한 맞춤 저녁 메뉴를 추천한다.

## 변경 파일 목록

### 백엔드
- `src/app/page.note.today/api.py`
  - `_get_my_children_info(user_id)` 헬퍼 함수 추가: 부모의 자녀 + 알레르기 + 나이 정보 조회
  - `recommend_dinner()` 개선:
    - 부모인 경우 자녀별 알레르기/나이/칼로리 기준 정보를 AI 프롬프트에 포함
    - 알레르기 있는 아이가 있으면 아이별 맞춤 추천 프롬프트 사용 (알레르기 식품 절대 제외)
    - 알레르기 없으면 기존 일반 추천 프롬프트 유지
  - `Users` 모델 import 추가
