# 일일 식단 진입 지연과 빈 화면 발생 원인 제거

## 작업 요약
캘린더 날짜 클릭과 일일 식단 좌우 이동 시 발생하던 지연을 줄이기 위해 일간 상세 데이터 요청을 캐시/프리패치 구조로 변경했다.
월간 데이터를 바탕으로 일간 화면을 즉시 표시하고, 실제 상세 응답은 공용 fetch 파이프라인으로 캐시한 뒤 덮어쓰도록 구성해 빈 화면과 재로딩 비용을 줄였다.
