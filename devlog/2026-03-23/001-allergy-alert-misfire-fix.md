# 알레르기 알림 오작동 수정

- **ID**: 001
- **날짜**: 2026-03-23
- **유형**: 버그 수정

## 작업 요약
알레르기 매칭 원생이 없을 때에도 교직원에게 불필요한 알림이 발송되는 문제를 수정했다. `_notify_staff_no_allergy()` 함수를 삭제하고, 알레르기 매칭 결과가 있을 때만 알림을 발송하도록 로직을 변경했다.

## 변경 파일 목록

### Route
- `src/route/api.scheduler/controller.py`: `_notify_staff_no_allergy()` 함수 삭제, 4곳의 호출부 제거, 알레르기 매칭 결과가 있을 때만 알림 발송하도록 조건 수정
