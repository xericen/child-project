# 어린이집 제공 열량 기준 변경

- **ID**: 010
- **날짜**: 2026-03-23
- **유형**: 기능 수정

## 작업 요약
열량 기준을 1일 전체 권장량(1~2세: 900kcal, 3~5세: 1400kcal)에서 어린이집 실제 제공분(점심+간식2회) 기준(1~2세: 420kcal, 3~5세: 640kcal)으로 변경. 비교 임계값도 45%→90%로 조정하고 UI 라벨을 "기준 충족/미달"로 업데이트.

## 변경 파일 목록

### 백엔드
- `src/app/page.note.meal/api.py`
  - `AGE_NUTRITION` kcal 값 변경: 900→420, 1400→640
  - `DEFAULT_TARGET_KCAL`: 1400→640
  - AI 분석 프롬프트: "연령별 1일 권장 영양 기준" → "어린이집 제공분 영양 기준", kcal 값 업데이트

### 프론트엔드
- `src/app/page.note.meal/view.ts`
  - `targetKcal` 초기값: 900→420
  - 임계값: `pct >= 45` → `pct >= 90`

- `src/app/page.note.meal/view.pug`
  - 요약 카드 라벨: "열량 45%↑/↓" → "기준 충족/미달"
  - 연령 선택기: "1일 {{ a.kcal }}kcal" → "{{ a.kcal }}kcal"
  - 캘린더 범례: "1일 권장 45% 이상/미만" → "기준 충족 (90%↑) / 기준 미달 (90%↓)"
